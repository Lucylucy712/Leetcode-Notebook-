# terminology used in Computer Science

## Elapsed real time 
[Source:Wiki](https://en.wikipedia.org/wiki/Elapsed_real_time)

Elpased real time, wall-clock time, or wall time is the actual time taken from the start of a computer program to the end. In other words, it's the difference between the time at which a task finished and the time at which the task started. 


## Immutable
A class is immutable if each obejct of that class has a fixed value upon instantiation that cannont subsequently be changed. For example, the float is immutable.
